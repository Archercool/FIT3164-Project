# Australian Weather Dashboard

## 📊 项目介绍

这个交互式天气仪表板是我与我的队友为Monash University FIT3164——数据科学项目所开发的毕业设计。其目标是创建一个直观、视觉丰富和可交互的平台，使用户能够探索澳大利亚的历史和近期天气趋势，包括温度和降雨量数据。仪表板为climate awareness, decision-making, 和 data exploration提供了宝贵的见解。

## 🌤️ 功能特性

### 实时天气
- 实时温度、湿度、风速、天气状况显示
- 支持多个澳大利亚城市（墨尔本、悉尼、布里斯班、珀斯、阿德莱德、堪培拉）
- 城市切换和位置定位功能

### 天气地图
- 交互式澳大利亚地图
- 温度和降雨量图层切换
- 彩色渐变显示天气模式
- 地理数据可视化和详细统计

### 历史数据分析
- 日温比较图表
- 降雨量分析
- 年度温度模式
- 极端天气统计
- 长期趋势分析

### 数据源
- Open-Meteo API 提供实时天气数据
- 内置澳大利亚各州地理数据

## 🛠️ 技术栈

### 前端
- HTML5
- CSS3
- JavaScript (ES6+)
- Tailwind CSS (通过CDN)
- Leaflet.js (地图功能)
- Chart.js (图表可视化)
- Font Awesome (图标)

### 后端
- Python Flask
- RESTful API
- SQLite Database

### 数据来源
- Open-Meteo API (免费天气数据)

- OpenStreetMap (地图底图)
